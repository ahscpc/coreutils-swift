# coreutils-swift
A work-in-progress project of rewriting the GNU Core Utilities in Swift

It is dependent on the `Foundation` and the `Darwin` or the `Glibc` Swift modules

## Commands that we have worked on
| Tool      | Completion    |
| --------- | ------------- |
| env       | **Done**      |
| ls        | 10 %          |
| uname     | 10 %          |
| sleep	    | **Done**      |
| whoami    | **Done**      |
